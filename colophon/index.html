Designers

The following staff of the Center for Instructional Innovation and Assessment contributed to the design, graphic creation, video production, and coding involved in the Showcase:

  • Graphic design: Wynter Thorne
  • Video footage and production: Sarah Edmonds, Justina Brown
  • Animated video introductions: Wynter Thorne
  • Scripting and CSS coding: Shahob Mousavi, Nickelos Hill
  • Web & Content development: Shahob Mousavi, Nickelos Hill, Andi Zamora, Hannah Chute, Sydney Nelson, Justina Brown

Design Elements

  • Typeface: Source Sans Pro - 300 (Light), 400 (Normal), and 700 (Bold)
    Fonts used are a part of Google Fonts
  • Colors: Western Blue (#003f87), #0083d6, #636780 (Abel), #4c7585 (Currier), #446e8f (Lee), Grey (#4d4e54), Almost Black (#202229).

Software Used

  • Adobe Photoshop / Illustrator
  • Atom IDE
  • Sublime Text 3
  • Adobe AfterEffects
  • Adobe Premiere

Coding

  • Based upon the Bootstrap HTML5/CSS/JS library
  • The jQuery JavaScript library
  • Embedded Videos - This Showcase utilizes YouTube to play movies on the "Videos" pages by embedding YouTube's proprietary code for each of our movies into the HTML.
